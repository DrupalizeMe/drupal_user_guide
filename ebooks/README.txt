This directory contains e-book output for the User Guide in all translated
languages. The file names end in the ISO language code for each
language. Formats currently available:

.pdf - Adobe's Portable Document Format.

.epub - Open-source ePub format.

.mobi - Amazon's Mobipocket format, also known as AZW, suitable for reading on
Kindle devices. Note that right-to-left languages, such as Farsi, are not
supported in this format.
