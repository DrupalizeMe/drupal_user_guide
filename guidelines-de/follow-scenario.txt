[[follow-scenario]]
=== Following the Scenario

[role="summary"]
Erklärt, wie Sie dafür sorgen, dass das project_name eine ganzheitliche Ausenwirkung behtält,
indem Sie sich an einem Szenario zur Erstellung einer Website orientieren, das durchgängig verwendet wird.

Das project_name folgt einem Szenario, das verwendet wird, um sicherzustellen, dass die behandelten Tehemen Einheilich dargestellt werden.
Behalten Sie das Szenario im Hinterkopf, wenn Sie Ihre Themen schreiben, insbesondere wenn
Sie Arbeitsaufgaben oder Konzepte hinzufügen (z.B. wenn Sie eine Aufgabe zum Erstellen eines Inhaltstyps auf einer
einer Website erstellen, sollte dieser Inhaltstyp zum von der Website verwendeten Szenario passen.
Szenario-Website).

Das Szenario können finden Sie im Vorwort des project_name Projekts.

In den folgenden Abschnitten erfahren Sie, wie Sie dem Szenario folgen können.


[[scenario-example]]
==== Verwenden Sie das Szenario als Beispiel

Wenn Sie eine Aufgabe oder ein Kapitel schreiben, geben Sie  Das Ziel,
die Anforderungen an die Seite, die Arbeitsschreitte, und eine Abschhnitt mit der Überschreift
„Verteifen Sie Ihr Wissen“ an, und verwenden dabei das Szenario als Beispiel.
 In einem Kapitel über das Erstellen von Inhaltstypen, fpr eine Website, in deren Szenario das
 Ersstellen einer Website für einen Wochenmarkt ist, könnte dies beispielsweise so aussehen:

Zielsetzung::
Erstellen Sie einen neuen Inhaltstyp, z.B. einen Lieferanten-Inhaltstypen für die Profilseiten der
Lieferanten des Wochenmarkts.
Arbeitsschritte::
Beschreiben Sie hier, wie das Formular zum Erstellen des Inhaltstyps auszufüllen ist und
geben Sie Beispielweerte an, mit denen das Formlar des Inhaltstyps auszufüllen ist.
Vertiefen Sie ihr Wissen::
  Erstellen Sie eine weiteren Inhaltstyp wie in diesem Handbuch beispielsweise den Inhaltstyp Rezepte.


[[scenario-profile]]
==== Gehen Sie davon aus, dass die vorherigen Aufgaben bereits ageschlossen wurden

Wenn Sie Aufgaben schreiben und Screenshots machen, gehen Sie davon aus, dass die vorherigen Aufgaben
auf der Website bereits abgeschlossen sind. Stellen Sie jedoch sicher, dass Sie die Anforderungen an die Website
, die notwendig sind, die Augfgabe erledigen zu können bereits gegeben sind. Z. B. müssen bestimmte Inhaltsstypen mit bestimmten Feldern bereits vorhanden sien. Außerdem kann es sien, dass ein paar Inhalte
eines bestimmten Typs schon vorhanden sein müssen, um das von Ihnen gewünschte Ergebnis zu erziehlen.


[[scenario-assets]]
==== Verwenden Sie die mitgeliegerten Assets

Das Projekt enthält verschiedene Assets (Texte, Bilder, und farbvorgaben). Die
Die Bilder sind lizenzkompatibel mit dem Projekt und können überall dort verwendet werden, wo
es angemessen ist. Verwenden Sie diese Assets wie folgt:

* Wenn Sie Beispielwerte für das Ausfüllen von Formularen bereitstellen, kopieren Sie diese und fügen Sie die
Text- und/oder Farbanweisungen in entsprechenden Felder wieder ein.

* Verwenden Sie beim Einrichten der Website zum Erstellen von Screenshots des Endergebnisses einer Aufgabe,
den bereitgestellten Text,  die vorgegebenen Farben und die mitgelieferten Bilder.

Die Assests Finden Sie im Verzeichnis project_cgit_assets_folder_link (Klicken Sie auf den Link
„plain" neben dem Dateinamen., um die Datei anzeign zu lassen). Sie auch:
