[[menu-reorder]]

=== Changer l'ordre de la navigation

[role="summary"]
Comment réordonner les liens dans un menu.

(((Liens de menu,réordonner)))
(((Navigation,changer l'ordre des liens de menus)))

==== Objectif

Réordonner les liens dans un menu.

==== Prérequis

* <<menu-concept>>
* <<menu-link-from-content>>

==== Prérequis du site

Les pages _Accueil_ et _À propos_ doivent exister dans le menu de navigation
principale. Consulter <<menu-link-from-content>>.

==== Étapes

. Dans le menu d'administration _Gérer_, aller à _Structure_ > _Menus_
(_admin/structure/menu_) où tous les menus de votre site sont listés. Cliquer
sur _Modifier le menu_ dans la liste d'_Opérations_ pour la _Navigation
principale_. Vous pouvez aussi accéder à cette page en utilisant les liens
contextuels (voir <<config-overview>>) pour le menu.
+
--
// Menu list section of admin/structure/menu, with Edit menu button on Main
// navigation menu highlighted.
image:images/menu-reorder_menu_titles.png["Liste des menus du site avec le menu
'Navigation principale' surligné",width="100%"]
--

. La page _Modifier le menu_ affiche une liste de chaque lien du menu choisi.
(_Navigation principale_).
+
--
// Menu links section of admin/structure/menu/manage/main.
image:images/menu-reorder_edit_menu.png["Liste des liens de menu sous
_Navigation principale_"]
--

. Faire glisser les liens de menus en utilisant la poignée en forme de croix
pour changer l'ordre : _Accueil_, puis _À propos_. Alternativement au
glisser/déposer, il est possible de cliquer sur le lien _Afficher les poids des
lignes_ en tête de tableau, et de sélectionner des poids numériques (les liens
de menus ayant des poids algébriques inférieurs seront affichés en premier).
+
--
// Menu links section of admin/structure/menu/manage/main, after
// changing the order.
image:images/menu-reorder_reorder.png["Liens de menu réordonnés avec le
message avertissant de modifications non enregistrées et le bouton _Enregistrer_"]
--

. Cliquer sur _Enregistrer_.

. La page d'accueil affiche maintenant la navigation principale avec le lien de
menu _Accueil_ affiché en premier.
+
--
// Header section of Home page with reordered menu links.
image:images/menu-reorder_final_order.png["Liens de menu réordonnés sur la page
d'accueil avec 'Accueil' en premier puis 'À propos"]
--

==== Améliorer votre compréhension

Ajouter un lien de menu appelé _Contact_, renvoyant ver la page de _/contact_,
dans votre menu de navigation principale. La page de contact est fournie par le
module du cœur Contact ; vous pouvez modifier sa mise en page et ses champs
(consulter <<structure-form-editing>>).
