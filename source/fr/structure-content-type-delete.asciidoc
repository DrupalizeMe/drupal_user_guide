[[structure-content-type-delete]]

=== Supprimer un type de contenu

[role="summary"]
Comment supprimer un type de contenu qui n'est pas nécessaire pour le site.

(((Type de contenu,supprimer)))

==== Objectif

Supprimer le type de contenu non nécessaire _Article_.

==== Pré-requis

<<config-overview>>

==== Pré-requis du site

Le type de contenu _Article_ doit exister. Il est créé sur votre site lorsque
vous l'installez avec le profil d'installation Standard du cœur.

==== Étapes

. Dans le menu d'administration _Gérer_, aller à _Structure_ > _Types de
contenu_ (_admin/structure/types_). La page des types de contenu apparaît.

. Cliquer sur le bouton _Supprimer_ dans la liste des _Opérations_ pour le type
de contenu _Article_.
+
--
// Content types list on admin/structure/types, with operations dropdown
// for Article content type expanded.
image:images/structure-content-type-delete-dropdown.png["Bouton déroulant
permettant de supprimer le type de contenu article",width="100%"]
--

. Une page de confirmation s'affiche. Cliquer sur _Supprimer_.
+
--
// Confirmation page for deleting Article content type.
image:images/structure-content-type-delete-confirmation.png["Page de
confirmation de la suppression du type de contenu article",width="100%"]
--

. La page des _types de contenu_ apparaît avec un message de confirmation disant
que le type de contenu a été supprimé :
+
--
// Confirmation message after deleting Article content type.
image:images/structure-content-type-delete-confirm.png["Message de confirmation
de la suppression du type de contenu article"]
--
