=== Створення аккаунта користувача

[role="summary"]
Як створити обліковий запис користувача.

(((Користувач, створення аккаунта)))

==== Мета

Створити аккаунт Постачальник для постачальників Солодкий мед і Веселий молочник.

==== Необхідні знання

* <<user-concept>>
* <<user-admin-account>>
* <<user-new-role>>

==== Вимоги до сайту

Роль Постачальник повинна існувати на вашому сайті. Дивіться <<user-new-role>>.

==== Кроки

. У _Керування_ адміністративного меню, перейдіть в _Люди_ (_admin/people_).

. Натисніть _Додати користувача_.
+
--
// Add new user form (/ admin/people/create).
image:images/user-new-user_form.png["Форма для створення нового облікового запису користувача", width="100%"]
--

. Заповніть поля форми. Дивіться на таблицю нижче.
+
[width="100%", frame="topbot", options="header"]
|================================
|Назва поля |Опис |Приклад значення
|Адреса електронної пошти |Дійсна адреса електронної пошти для постачальника. Всі електронні листи з системи будуть відправлені на цю адресу. Адреса електронної пошти не оприлюднюється. |honey@example.com
|Ім'я користувача (Username) |Ім'я користувача для постачальника, яке вони будуть використовувати для входу або створення матеріалів. Пробіли дозволені; пунктуація не допускається, за винятком крапок, дефісів, апострофів і підкреслень. |Солодкий мед
|Пароль |Пароль, який постачальник буде використовувати для входу на сайт. Ви можете побачити, наскільки безпечний пароль, на _Індикаторі складності_ пароля. Ви також отримаєте поради про те, як зробити його безпечнішим. |(Зробіть надійний пароль)
|Повторіть пароль |Введіть той же пароль, щоб уникнути помилок при наборі. |(Повторіть пароль)
|Стан |Встановіть стан призначеного для користувача аккаунта. _Заблоковано_ користувачі не зможуть увійти. |Активно
|Ролі (Roles) |Встановіть роль призначеного для користувача аккаунта. |Постачальник
|Повідомити користувача про створення облікового запису |Чи варто відправляти повідомлення на електронну адресу постачальника. |Обрано
|Зображення |Натисніть _Choose file_ і виберіть зображення для завантаження. Зверніть увагу на обмеження за розміром. |Фотографія постачальника
|Налаштування контактної форми |Включити або відключити відображення контактної форми для аккаунта. |Обрано
|================================


. Натисніть _Реєстрація_. Ви отримаєте повідомлення про створення призначеного для користувача
аккаунта.
+
--
// Confirmation message after adding new user.
image:images/user-new-user-created.png["Новий акаунт користувача був створений", width="100%"]
--

. Створіть другий аккаунт Постачальник для Веселого молочника, виконавши дії, описані вище.

==== Поліпшіть своє розуміння

Створіть акаунт користувача для себе.
