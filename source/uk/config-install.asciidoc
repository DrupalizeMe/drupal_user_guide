==== Вимоги до сайту

==== Кроки

Ви можете використовувати адміністративний інтерфейс або Drush для установки модулів.

===== Використання адміністративного інтерфейсу

. В адміністративному меню _Упаравління_, перейдіть в _Розшрення_
(_admin/modules_). На сторінці _Розширення_ показуються всі доступні модулі
на вашому сайті.

. Проставте галочки для модуля або модулів, які ви хочете встановити. Наприклад,
поставте галочку для модуля ядра Activity Tracker.
+
--
// Top part of Core section of admin / modules, with Activity Tracker checked.
image:images/config-install-check-modules.png["Включення модуля Activity Tracker", width = "100%"]
--

. Натисніть _Встановити_. Вибрані модулі будуть встановлені.

===== Використання Drush

. У розділі _Керування_ адміністративного інтерфейсу, перейдіть в _Розширення_
(_admin/modules_). На сторінці _Розширення_ показані всі встановлені модулі
на вашому сайті.

. Знайдіть машинне ім'я модуля який ви хочете встановити, розкривши
блок інформації для потрібного модуля. Наприклад, Activity Tracker
модуль має машинне ім'я _tracker_.

. Запустіть наступну Drush команду, щоб встановити потрібний модуль:
+
----
drush pm:enable tracker
----

==== Поліпшіть своє розуміння

Якщо ви не спостерігаєте змін на вашому сайті, можливо вам потрібно
почистити кеш. Дивіться <<prevent-cache-clear>>.
