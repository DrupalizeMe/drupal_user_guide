=== Налаштування теми

[role="summary"]
Як редагувати налаштування теми, щоб оновити схему кольорів та додати логотип.

((((Тема, налаштування)))
((((Тема Bartik, налаштування)))
((((Схема кольорів, налаштування)))
((((Зображення логотипу, налаштування)))

==== Мета

Змінити налаштування основної теми Bartik за замовчуванням, щоб змінити схему кольору
та додати логотип.

==== Необхідні знання

<<understanding-themes>>

// ==== Передумови веб-сайту

==== Кроки

. В адміністративному меню _Керування_ перейдіть до _Зовнішній вигляд_
(_admin/appearance_).

. У розділі _Встановлені теми_ ви знайдете Bartik як базова тема. У розділі _Bartik (базова тема)_ клацніть _Налаштування_.
+
--
// Розділ Bartik admin/appearance.
image:images/config-theme_bartik_settings.png["Тема за замовчуванням, Bartik", width="100%"]
--

. У розділі _Схема кольорів_ клацніть всередині кожного поля вибору кольору та введіть відповідні
коди кольорів, які ви хотіли б додати. Наприклад, використовуйте
наступні кольори:
+
[width="100%", frame="topbot", options="header"]
|==================================
|Площа |Колір
|Заголовок фона зверху |# 7db84a (зелений)
|Заголовок фона знизу |# 2a3524 (темно-зелений)
|Основна довідка |#ffffff (білий)
|Фон бічної панелі |# f8bc65 (світло-оранжевий)
|Межі бічної панелі |# e96b3c (оранжевий)
|Фон нижнього колонтитула |# 2a3524 (темно-зелений)
|Назва та гасло |#ffffff (білий)
|Колір тексту |# 000000 (чорний)
|Колір посилання |# 2a3524 (темно-зелений)
|==================================
+
Примітка: Ви також можете скористатися кольоровим кружком праворуч, щоб вибрати кольори на ваш вибір. Коди веб-кольорів будуть додані для вас.
+
--

// Розділ налаштувань кольору admin/appearance/settings/bartik.
image:images/config-theme_color_scheme.png["Оновлені специфікації кольорів в основній темі Bartik", width="100%"]
--

. У розділі _Зображеня логотипа_ зніміть прапорець _Використовувати логотип, пропонований темою_.
+
--
// Розділ завантаження логотипу адміністратора / зовнішнього вигляду / налаштувань / bartik.
image:images/config-theme_logo_upload.png["Новий логотип у основних налаштуваннях теми Bartik"]]
--

. У розділі _Викласти зображення з логотипом_ знайдіть файл логотипу та завантажте його у свій
сайт. Примітка. Ви також можете встановити універсальний логотип для всіх тем
_Вигляд_ > _Налаштування_ (_admin/appearance/settings_). Спеціальний логотип для вашої
теми замінить універсальний логотип.
+
Вибравши файл, який ви хочете завантажити, ви побачите його
ім'я файлу поруч із кнопкою _Вибрати файл_ або _Переглянути_ у вашому браузері.

. Щоб зберегти зміни та побачити оновлені кольори та логотип на
веб-сайті, натисніть _Зберегти конфігурацію_ внизу сторінки.
+
Примітка: У розділі _Схема кольорів_ є розділ _Переглянути_, який відображає зразок
про те, як ваш веб-сайт буде виглядати з новими налаштуваннями.
+
--
// Розділ попереднього перегляду admin/appearance/ settings/bartik.
image:images/config-theme_color_scheme_preview.png["попередній перегляд налаштувань основної теми Bartik", width="100%"]
--

. Клацніть _Повернутися на сайт_ або _Додому_ на панелі інструментів, щоб підтвердити, що система
оновила основні налаштування теми Bartik для вашого веб-сайту.
+
--
// Домашня сторінка після завершення налаштування теми.
image:images/config-theme_final_result.png["Остаточний результат редагування основних налаштувань теми Bartik - кольори та логотип", width="100%"]
--
