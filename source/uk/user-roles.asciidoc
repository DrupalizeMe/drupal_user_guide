=== Зміна ролей користувача

[role="summary"]
Як вибрати або додати роль для користувача.

(((Роль користувача, вибір)))
(((Роль, вибір)))
(((Дозвіл, вибір ролі)))

==== Мета

Зміна або додавання ролей для даного користувача шляхом редагування одного користувача або
застосування масової операції.

==== Необхідні знання

<<user-concept>>

==== Вимоги до сайту

Обліковий запис користувача, який ви хочете оновити, і роль, яку ви хочете надати, повинні
існувати. Дивіться <<user-new-user>>, <<user-new-role>>, і
<<user-permissions>>.

==== Кроки

===== Оновлення ролей з використанням розрахованого на одного користувача методу редагування

. У _Керування_ адміністративного меню, перейдіть в _Люди_
(_admin/people_).

. Знайдіть акаунт користувача 1 (ім'я "admin") з призначеною роллю _Адміністратор_.
Якщо його не відразу видно, використовуйте фільтр _Ім'я чи електронна адреса містять_,
або інші фільтри, щоб звузити список.

. Натисніть _Редагуати_, щоб оновити обліковий запис користувача.
+
--
// People page (admin/people), with user 1's Edit button outlined.
image:images/user-roles_people-list.png["Сторінка користувачів", width="100%"]
--

. На сторінці _Редагувати_, перейдіть до розділу _Ролі_. Виберіть роль _Адміністратор_
для аккаунта користувача.
+
--
// Roles area on user editing page.
image:images/user-roles_person-edit.png["Розділ ролей сторінки редагування користувача", width="141px"]
--

. Натисніть _Зберегти_, щоб оновити обліковий запис користувача. Ви будете повернуті на
сторінку _Люди_ і побачите повідомлення про те, що зміни були збережені.
+
--
// Confirmation message after updating user.
image:images/user-roles_message.png["Повідомлення підтвердження"]
--

===== Оновлення ролей методом масового редагування

. Якщо у користувачів Веселий молочник і Алтайський мед ще не було ролі Постачальник,
ось як потрібно додати цю роль. У _Керування_ адміністративного меню, перейдіть в
_Люди_ (_admin/people_).

. Знайдіть облікові записи Постачальника _Солодкий мед_ і _Веселий молочник_ і перевірте їх. Якщо
їх не відразу видно, використовуйте фільтр _Ім'я чи електронна адреса містять_,
або інші фільтри, щоб звузити список.

. Виберіть _Додати роль Постачальник для обраних користувачів_ зі списку _Дія_.
+
--
// Bulk editing form on People page (admin/people).
image:images/user-roles_bulk.png["People page with bulk operation set up", width="100%"]
--

. Натисніть _Застосувати до вибраних позицій_. Ви повинні побачити повідомлення про те, що
бажані зміни були зроблені.
+
--
// Confirmation message after bulk user update.
image: images/user-roles_message_bulk.png [ "Confirmation message"]
--
