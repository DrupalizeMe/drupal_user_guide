[[block-create-custom]]

=== Creating A Custom Block

[role="summary"]
How to create a custom block.

(((Block,creating)))
(((Custom block,creating)))

==== Goal

Create a block showing the hours and location of the farmers market.

==== Was man wissen sollte

* <<block-concept>>
* <<block-regions>>

// ==== Site prerequisites

==== Steps

. In the _Manage_ administrative menu, navigate to _Structure_ > _Block
layout_ > _Custom block library_ (_admin/structure/block/block-content_).

. Click _Add custom block_. The _Add custom block_ page appears.

. Fill in the fields as shown below.
+
[width="100%",frame="topbot",options="header"]
|================================
|Field name |Explanation |Example value
|Block description |Name of the block shown to administrators |Hours and location block
|Body |Content of the block when it is displayed |Open: Sundays, 9 AM to
2 PM, April to September<br />Location: Parking lot of Trust Bank, 1st & Union,
downtown Anytown.
|================================
+
--
// Block add page (block/add).
image:images/block-create-custom-add-custom-block.png["Add a custom block", width="100%"]
--

. Click _Save_. A message appears indicating the block has been saved.

==== Expand your understanding

* Edit the content of your custom block. In the _Manage_ administrative menu,
navigate to _Structure_ > _Block layout_ > _Custom block library_
(_admin/structure/block/block-content_). Find your block in the list and click
_Edit_ to make changes.

* Place the block you created in the sidebar. See <<block-place>> for details.
