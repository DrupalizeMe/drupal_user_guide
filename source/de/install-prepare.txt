[[install-prepare]]
=== Vorbereiten der Installation

[role="summary"]
Das Kernsystem herunterladen und die Installationsvoraussetzungen erfüllen.

(((Downloading,core software)))
(((Drupal core,preparing install)))

==== Zielsetzungsetzung

Laden Sie die Kern-Software herunter und kümmern Sie sich um alle erforderlichen Voraussetzungen.

==== Vorgeschriebene Kenntnisse

* <<install-requirements>>

* <<install-tools>>

* <<install-decide>>

* Wie man Server-Software installiert und konfiguriert (falls Sie die Installation selbst vornehmen
Computer)

* Wie man das Hosting für eine generische Website einrichtet

* Wie man eine Datenbank erstellt

==== Standort-Voraussetzungen

Die Server-Software muss auf dem Computer installiert sein, auf dem Sie Ihre
Standort. Siehe <<installierte Anforderungen>>.

Je nachdem, wie Sie die Kernsoftware herunterladen möchten, müssen Sie möglicherweise Folgendes installieren
zusätzliche Software-Tools zuerst. Siehe <<Installieren-Entscheiden>> und <<Installieren-Werkzeuge>>.

==== Arbeitsschritte

. Wählen Sie unter den folgenden Methoden zum Herunterladen und Installieren der Kern-Software
aufgelistet unter <<install-decide>>. Der Rest dieser Anweisungen gilt für die
Download-Optionen für den Komponisten und das Handbuch sowie das interaktive Installationsprogramm; wenn Sie
andere Optionen, sollte die Software für Sie installiert werden.

. Richten Sie eine URL und ein Hosting für Ihre Website auf dem Server ein. Überprüfen Sie, ob das Hosting
funktioniert, indem es eine einfache HTML-Datei in das Web-Stammverzeichnis der
Hosting und Besuch der URL für Ihre Website.

. Erstellen Sie eine Datenbank, zusammen mit einem Datenbank-Benutzerkonto mit Vollzugriff.

. Laden Sie die Kern-Software-Dateien in das Web-Stammverzeichnis herunter, indem Sie die Methode
für die Sie sich entschieden haben. Siehe <<installieren-entscheiden>> für Links zu Anweisungen.

==== Erweitern Sie Ihr Verständnis

Siehe <<install-run>> um das interaktive Installationsprogramm auszuführen.

Alternativ können Sie den folgenden Drush-Befehl aus dem
Verzeichnis, in das Sie die Software heruntergeladen haben, wobei _DB_NAME_, _DB_USER_ und
_DB_PASS_ sind die Anmeldedaten Ihrer Datenbank:

----
drush site:install standard --db-url='mysql://DB_USER:DB_PASS@localhost/DB_NAME' --site-name=example
----
