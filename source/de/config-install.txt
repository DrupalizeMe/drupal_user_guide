[[config-install]]
=== Ein Modul installieren

[role="summary"]
Ein Modul des Kernsystems oder ein von der Community bereitgestelltes Modul über die Benutzeroberfläche installieren.
or Drush.

(((Module,enabling)))
(((Module,installing)))

==== Ziel

Installieren Sie ein mit dem Kernsystem ausgeleifertes Modul oder einvon der Community bereitgestelltes Zusatzmodul, dass bereits auf den Webserver
hochgeladen wurden, mit Hilfe des Verwaötungsbereichs oder mittels Drush.

==== Erforderliche Vorkenntnisse

<<understanding-modules>>

==== Anforderungen an die Website

Wenn Sie Drush zur Installation von Modulen verwenden möchten, muss Drush installiert sein. Siehe:
<<install-tools>>.

==== Arbeitsschritte

Sie können den Verwaltungsbereich oder Drush verwenden, um Module zu installieren.

==== Verwendung der administrativen Schnittstelle

. Navigieren Sie im Menü _Verwalten_ zum Menüpunkt _Erweitern_
(_admin/modules_). Die Seite _Erweitern_ erscheint und zeigt alle auf der Website verfügbaren Module an.

. Markieren Sie die Kontrollkästchen für das Modul oder die Module, die Sie installieren möchten. Markieren Sie zum Beispiel
das Kontrollkästchen für das mit dem Kernsystem ausgelieferten Modul Activity Tracker.
+
--
// Top part of Core section of admin/modules, with Activity Tracker checked.
image:images/config-install-check-modules.png["Enabling the core Activity Tracker module",width="100%"]
--

. Klicken Sie auf _Installieren_. Die markierten Module werden installiert.

==== Drush verwenden

. Navigieren Sie im Menü _Verwalten_ zum Menüpunkt _Erweitern_
(_admin/modules_). Die Seite _Erweitern_ erscheint und zeigt alle auf der Website verfügbaren Module an.

. Suchen Sie den Systemnamen des Moduls, das Sie installieren möchten, indem Sie den
Informationsbereich für das Modul. Das Modul Activity Tracker,
das mit dem Kernsystem ausgeliefert wird, trägt beispielsweise den Systemnamen _tracker_.

. Führen Sie den folgenden Drush-Befehl aus, um das Modul zu installieren:
+
----
drush pm:enable tracker
----

==== Vertiefen Sie Ihr Wissen

Wenn Sie die Auswirkungen dieser Änderungen auf Ihrer Website nicht sehen, müssen Sie zunächst den Cache leeren.
Siehe: <<prevent-cache-clear>>.
