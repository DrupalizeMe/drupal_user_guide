[[structure-content-type-delete]]

=== Einen Inhaltstyp löschen
[role="summary"]
So löschen Sie einen Inhaltstyp, der für die Site nicht erforderlich ist.

(((Content type,deleting)))

Zielsetzung

Löschen Sie den nicht benötigten Inhaltstyp _Artikel_.

==== Vorausgesetztes Wissen

<<config-overview>>

==== Standort-Voraussetzungen

Der Inhaltstyp _Artikel_ muss vorhanden sein. Er wird auf Ihrer Site angelegt, wenn Sie
mit dem Kern-Standardinstallationsprofil installieren.

==== Schritte

. Navigieren Sie im Verwaltungsmenü _Verwaltung_ zu _Struktur_ > _Inhalt
Typen_ (_admin/structure/types_). Die Seite _Inhaltstypen_ erscheint.

. Klicken Sie auf _Löschen_ in der Dropdown-Schaltfläche _Operationen_ für den Inhalt _Artikel_.
tippen. Beachten Sie, dass der Name dieses Inhaltstyps auf dieser Seite auf Englisch ist;
siehe <<language-concept>> for an explanation.
+
--
// Content types list on admin/structure/types, with operations dropdown
// for Article content type expanded.
image:images/structure-content-type-delete-dropdown.png["Article delete dropdown",width="100%"]
--

. Eine Bestätigungsseite wird angezeigt. Klicken Sie auf _Löschen_..
+
--
// Confirmation page for deleting Article content type.
image:images/structure-content-type-delete-confirmation.png["Article delete confirmation page",width="100%"]
--

. Die Seite _Inhaltstypen_ erscheint mit einer Bestätigungsmeldung, die besagt, dass die
Content-Typ wurde gelöscht:
+
--
// Confirmation message after deleting Article content type.
image:images/structure-content-type-delete-confirm.png["Article delete confirmation message"]
--
