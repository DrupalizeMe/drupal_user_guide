[[structure-content-type-delete]]

=== Einen Inhaltstyp löschen
[role="summary"]
So löschen Sie einen Inhaltstyp, den Sie auf Ihrer Website nicht mehr benötigten.

(((Content type,deleting)))

==== Zielsetzung

Löschen Sie den nicht benötigten Inhaltstyp _Artikel_.

==== Erforderliche Vorkentnisse

<<config-overview>>

==== Anforderungen an die Website

Der Inhaltstyp _Artikel_ muss vorhanden sein. Er wird auf Ihrer Website angelegt, wenn diese
mit dem installitionsprofil Standard intalliert wird, das mit dem Kernsystem ausgeliefert wird.

==== Arbeitsschritte

. Navigieren Sie im Menü _Verwalten_ zum Menüpunkt _Struktur_ > _Inhaltstypen_
(_admin/structure/types_). Die Seite _Inhaltstypen_ erscheint.

. Klicken Sie im Drop-Button in der Spalte _Aktionen_ hinter den Inhaltstyp Artikel auf _Löschen_.
Beachten Sie, dass der Name dieses Inhaltstyps auf dieser Seite auf Englisch angezeigt wird;
siehe <<language-concept>> für weitere Konfigurationen.
+
--
// Content types list on admin/structure/types, with operations dropdown
// for Article content type expanded.
image:images/structure-content-type-delete-dropdown.png["Article delete dropdown",width="100%"]
--

. Eine Bestätigungsseite wird angezeigt. Klicken Sie auf _Löschen_.
+
--
// Confirmation page for deleting Article content type.
image:images/structure-content-type-delete-confirmation.png["Article delete confirmation page",width="100%"]
--

. Die Seite _Inhaltstypen_ wird geöffnet und es wird eine Bestätigungsmeldung ausgegeben, die besagt, dass der Inhaltstyp Artikel gelöscht wurde:
+
--
// Confirmation message after deleting Article content type.
image:images/structure-content-type-delete-confirm.png["Article delete confirmation message"]
--
