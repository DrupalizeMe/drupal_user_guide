[[menu-reorder]]

=== Ändern der Reihenfolge der Navigation

[role="summary"]
Wie man Elemente in einem Menü nü anordnet.

(((Menu items,reordering)))
(((Navigation,changing the order of menu items)))

===== Ziel

Ordnen Sie die Einträge in einem Menü neu an.

===== Vorkenntnisse

* <<menu-concept>>
* <<menu-link-from-content>>

===== Standort-Voraussetzungen

Die Seiten Home und About müssen im Hauptnavigationsmenü vorhanden sein. Siehe
<<<Menü-Link-vom-Inhalt>>.

===== Schritte

. Navigieren Sie im Verwaltungsmenü _Verwaltung_ zu _Struktur_ > _Menüs_
(_Admin/Struktur/Menü_), wo alle Menüs auf Ihrer Website aufgelistet sind. Klicken Sie auf _Bearbeiten
Menü_ aus der Dropdown-Liste _Operations_ für die _Hauptnavigation_. Sie können auch erreichen
diese Seite mit kontextuellen Links (siehe <<Config-Übersicht>>>) für die
Menü. Beachten Sie, dass die Namen und Beschreibungen der Menüs, die von
Ihr Installationsprofil wird auf dieser Seite auf Englisch angezeigt; siehe
<<language-concept>> for an explanation.
+
--
// Menu list section of admin/structure/menu, with Edit menu button on Main
// navigation menu highlighted.
image:images/menu-reorder_menu_titles.png["List of menus on the site with 'Main navigation' highlighted",width="100%"]
--

. Die Seite _Bearbeiten-Menü_ zeigt eine Liste aller Einträge im Menü an, die Sie
gewählt (_Hauptnavigation_).
+
--
// Menu links section of admin/structure/menu/manage/main.
image:images/menu-reorder_edit_menu.png["List of menus items under _Main navigation_"]
--

. Ziehen Sie die Querbalkengriffe der Menüelemente, um ihre Reihenfolge zu ändern: Nach Hause, dann
Über. Als Alternative zum Ziehen können Sie auf den Link _Zeilengewichte anzeigen_ klicken
oben in der Tabelle und wählen Sie numerische Gewichte (Menüpunkte mit niedrigerer oder
weitere negative Gewichte werden zuerst angezeigt).
+
--
// Menu links section of admin/structure/menu/manage/main, after
// changing the order.
image:images/menu-reorder_reorder.png["Reordered menu items with warning message about unsaved changes and the _Save_ button"]
--

. Klicken Sie auf _Speichern_.

. Die Startseite zeigt nun die Hauptnavigation mit dem Menü Home
Artikel wird zuerst angezeigt.
+
--
// Header section of Home page with reordered menu items.
image:images/menu-reorder_final_order.png["Reordered menu items on the home page with 'Home' first and then 'About'"]
--

===== Erweitern Sie Ihr Verständnis

Fügen Sie einen Menüpunkt namens _Kontakt_, der zur Seite _/Kontakt_ führt, zu Ihrem Hauptmenü
Navigationsmenü. Die Kontaktseite wird durch das Kernmodul Kontakt bereitgestellt; Sie
kann sein Layout und seine Felder bearbeiten wollen (siehe <<Struktur-Form-Bearbeitung>>).
