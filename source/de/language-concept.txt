==== Welche Sprachen unterstützt die Software?

Die Basissprache für die Software, die auf Ihrer Website läuft (Kern-Software, Module,
und Themen) ist englisch. Mit dieser Software können Sie jedoch eine Website erstellen, deren
Die Standardsprache ist nicht Englisch. In diesem Fall sollte jeder, der die Website besucht
nur diese Sprache sehen (unter der Annahme, dass die Website vollständig übersetzt ist). Sie können
Verwenden Sie diese Software auch, um eine mehrsprachige Website mit einem Sprachumschalter zu erstellen.
mit der Besucher der Website zu ihrer bevorzugten Sprache wechseln können. Sie müssen
das Kern-Sprachmodul installiert haben, um eine andere Sprache als
Englisch auf der Website.

==== Was kann auf Ihrer Website übersetzt werden?

Es gibt drei Arten von Informationen, die Sie übersetzen können, jede mit ihren eigenen
Methode zum Übersetzen:

Text der Benutzungsoberfläche::
  Eingebauter Text, der in der Kern-Software, den Modulen und Themen vorhanden ist. Dies kann sein
  aus der englischen Basissprache der Software in die Sprache(n) übersetzt
  Ihrer Website. Normalerweise müssen Sie diesen Text nicht selbst übersetzen,
  können Sie Übersetzungen herunterladen. Sie benötigen den Kern Interface Übersetzung
  Modul installiert, um diesen Text zu übersetzen, und der zentrale Update-Manager
  Modul installiert, um Übersetzungen automatisch herunterzuladen.
Text der Konfiguration::
  Text, dessen Struktur und Anfangswerte durch die Kern-Software definiert werden,
  Module und Themen, die Sie jedoch bearbeiten können. Beispiele sind die Bezeichnungen für
  Felder in Ihren Inhaltstypen, Kopftext in Ansichten, Ihren Site-Namen und die
  Inhalt der automatischen E-Mail-Nachrichten, die Ihre Website versendet. Nach der Erstellung
  Konfigurationstext in der Standardsprache Ihrer Website, Sie können ihn übersetzen
  in andere Sprachen. Für die von der Kern-Software gelieferte Standardkonfiguration,
  Module und Themen, die Übersetzung ist in den Downloads der Benutzer enthalten
  Übersetzungen von Schnittstellentexten. Sie benötigen den Kern Konfiguration
  Übersetzungsmodul installiert, um diesen Text zu übersetzen.
Inhaltliche Texte und Dateien::
  Wenn Ihre Website mehrsprachig ist, können Sie die Inhaltsfelder auf Ihrer
  Website übersetzbar sein. Nachdem Sie Inhalte in einer Sprache erstellt haben, können Sie
  sie in andere Sprachen übersetzen. Felder können Textinformationen enthalten oder
  hochgeladenen Dateien, und für jedes Feld auf jedem Inhaltstyp können Sie es konfigurieren
  übersetzbar oder nicht übersetzbar zu sein. Sie benötigen den Kerninhalt
  Übersetzungsmodul installiert, um diesen Text zu übersetzen.

==== Welche Informationen werden auf meiner Website auf Englisch bleiben?

Selbst wenn die Standardsprache Ihrer Website nicht Englisch ist, sehen Sie
Englischer Text auf bestimmten administrativen Seiten, die zur Verwaltung der Konfiguration verwendet werden. Die
Der Grund dafür ist, dass Sie beim Bearbeiten der Konfiguration die Basis bearbeiten,
unübersetzte Konfigurationswerte; das Übersetzen der Konfiguration ist eine separate
Betrieb. Wenn Sie zum Beispiel die Verwaltungsseite _Menüs_ besuchen, werden Sie
siehe Menünamen auf Englisch (für die Menüs, die bei der Installation eingerichtet wurden
Ihre Website), und wenn Sie auf einen _Bearbeiten-Menü_-Link klicken, bearbeiten Sie die
Englischer Name und Beschreibung des Menüs. So bearbeiten Sie den Menünamen in einem anderen
Sprache benötigen Sie das Kernmodul Configuration Translation
installiert, und verwenden Sie den Link _Translate_, um die übersetzten Menüinformationen zu bearbeiten.
