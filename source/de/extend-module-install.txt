[role="summary"]
Wie man ein Modul von Drupal.org herunterlädt und installiert (über die Benutzeroberfläche)

Zielsetzung

Herunterladen und Installieren der
https://www.drupal.org/project/admin_toolbar[beigetragenes Admin Toolbar-Modul],
die es Ihnen ermöglicht, einfach durch den Verwaltungsbereich der
Website.

==== Erforderliche Vorkenntnisse

* <<understanding-modules>>
* <<extend-module-find>>
* <<install-tools>>

==== Anforderungen an die Website

* Wenn Sie Module über die Website installieren möchten, muss das Kernmodul Update Manager
muss installiert sein. Siehe <<Config-install>> für Anweisungen zur Installation
Module.

* Wenn Sie Drush verwenden möchten, muss Drush installiert sein. Siehe <<<installieren-Werkzeuge>>.

==== Schritte

Sie können die administrative Schnittstelle zum Herunterladen und Installieren einer
Modul. Wenn Sie ein benutzerdefiniertes Modul anstelle eines beigetragenen Moduls installieren,
wenn Sie eine Meldung sehen, die besagt: _Die Installation von Modulen und Themen erfordert FTP-Zugang
auf Ihren Server_, oder wenn die folgenden Schritte nicht funktionieren, laden Sie die
Moduldateien, folgen Sie den Schritten unter <<erweitern-manual-install>>. Wenn Sie
Composer, um die Dateien auf Ihrer Website zu verwalten, folgen Sie den Schritten zum Herunterladen in
<<Installieren-Komponist>>. In jedem dieser Fälle können Sie dann das Modul
mit Drush, oder indem Sie mit Schritt 7 in der Anleitung für die
administrative Schnittstelle unten.

======= Verwendung der administrativen Schnittstelle

. Auf der _Admin Symbolleiste_ Projektseite auf drupal.org
(_https://www.drupal.org/project/admin_toolbar_), blättern Sie zu den _Downloads_
Abschnitt am Ende der Seite.

. Kopieren Sie die Adresse des _tar.gz_-Links. Abhängig von Ihrem Gerät und Browser,
können Sie dies tun, indem Sie mit der rechten Maustaste klicken und _Linkadresse kopieren_ auswählen.
+
--
// Downloads-Abschnitt der Admin Toolbar Projektseite auf drupal.org.
image:images/extend-module-install-download.png["Eine Modul-URL finden"]
--

. Navigieren Sie im Verwaltungsmenü _Verwalten_ zu _Erweitern_
(_Verwaltung/Module_). Die Seite _Erweitern_ erscheint.

. Klicken Sie auf _Neues Modul installieren_. Die Seite _Neues Modul _Installieren_ erscheint.
+
--
// Neue Modulseite installieren (admin/modules/install).
image:images/dextend-module-install-admin-toolbar-do.png["Ein Modul von einer URL installieren"]
--

. Fügen Sie in das Feld _Install from a URL_ den kopierten Download-Link ein. Dieser Wert
könnte so aussehen:
{\an8}COPY15

. Klicken Sie auf _Install_, um das neue Modul auf den Server hochzuladen und zu entpacken. Die Dateien
werden in das Verzeichnis _modules_ heruntergeladen.

. Klicken Sie auf _Neu hinzugefügte Module aktivieren_, um zur Seite _Erweitern_ zurückzukehren. Wenn Sie
das bereits erwähnte manuelle Hochladeverfahren verwendet haben, dann können Sie fortfahren
mit diesem Schritt, und erreichen Sie die Seite _Erweitern_ mit der Funktion _Verwalten_
Verwaltungsmenü und Navigation zu _Erweitern_ (_Admin/Module_).

. Suchen Sie das Modul _Admin-Symbolleiste_ und überprüfen Sie es.

. Klicken Sie auf _Install_, um das neue Modul einzuschalten.

====== Drush zum Installieren eines Moduls verwenden

. Suchen Sie den Projektnamen des Moduls, das Sie installieren möchten, d.h. den letzten
Segment der URL der Projektseite des Moduls. Wenn die Projekt-URL beispielsweise
_https://www.drupal.org/project/admin_toolbar_, der Projektname lautet
"admin_toolbar".

. Laden Sie das Modul herunter, indem Sie entweder die Schritte unter <<erweitert-manual-install>> oder
<<Installieren-Komponist>>.

. Führen Sie den folgenden Drush-Befehl unter Angabe des Projektnamens aus (z.B,
`admin_toolbar`) als Parameter:
+
----
drush pm:enable admin_toolbar
----

. Folgen Sie den Anweisungen auf dem Bildschirm.

==== Erweitern Sie Ihr Verständnis

* Überprüfen Sie, ob die
https://www.drupal.org/project/admin_toolbar[beigetragenes Admin Toolbar-Modul]
arbeitet, indem sie durch das Menü im Verwaltungsbereich blättert.

* Installieren und konfigurieren Sie die
https://www.drupal.org/project/pathauto[beigetragenes Pathauto-Modul]
so dass Inhaltsseiten in Ihrer Website standardmäßig schöne URLs erhalten. Siehe
<<<Inhaltspfade>> für mehr zu URLs.

* Wenn Sie die Auswirkungen dieser Änderungen auf Ihre Website nicht sehen, benötigen Sie möglicherweise
um den Cache zu leeren. Siehe <<<Cache-Löschen verhindern>>.
