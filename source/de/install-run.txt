=== Ausführen des interaktiven Installationsprogramms

[role="summary"]
Das interaktive Installationsprogramm zur Installation des Kernsystems verwenden.

==== Zielsetzungsetzung

Installieren Sie die Kern-Software und erstellen Sie das Administratorkonto, indem Sie die mitgelieferte
Installationsprogramm.

==== Standort-Voraussetzungen

<<install-prepare>>

==== Arbeitsschritte

. Wenn Sie eine 1-Klick-Installation von einem Hosting-Provider oder einer Demo-Site verwenden, können Sie
werden höchstwahrscheinlich einige oder alle der folgenden Bildschirme als Teil der
Installationsprozess. Wenn Sie die Kerndateien manuell oder mit dem Composer hochgeladen haben,
um das Installationsprogramm zu starten, öffnen Sie einen Browser und besuchen Sie die URL, die Sie für
Ihr Hosting.

. Wählen Sie zum Beispiel auf der ersten Seite des Installationsprogramms eine Sprache aus,
Englisch. Sie können optional eine der anderen aufgeführten Sprachen wählen.
Die Sprachdateien für die gewählte Sprache werden heruntergeladen und installiert, so
dass der Rest des Installationsprozesses in der gewählten
Sprache. Nachdem Sie eine Sprache ausgewählt haben, klicken Sie auf _Speichern und fortfahren_.
+
--
// Go to the first step of the installer
image:images/install-run-1.png["Choose a language",width="100%"]
--

. Wählen Sie ein Installationsprofil aus. Installationsprofile bieten Standortmerkmale
und Funktionen für eine bestimmte Art von Website als ein einziger Download, der
die Kern-Software, beigetragene Module, Themen und vordefinierte
Konfiguration. Der Kern enthält zwei Installationsprofile. Wählen Sie den Core aus
Standard-Installationsprofil. Klicken Sie auf _Speichern und fortfahren_.
+
--
// Go to the choose an installation profile step of the installer
image:images/install-run-2.png["Choose an installation profile",width="100%"]
--

. Im nächsten Schritt des Installationsprogramms wird überprüft, ob Ihr System die Mindestvoraussetzungen erfüllt
Anforderungen. Wenn dies nicht der Fall ist, erhalten Sie einen Überblick darüber, was
muss korrigiert werden, um fortfahren zu können. Wenn dies der Fall ist, wird das Installationsprogramm
automatisch zum nächsten Schritt übergehen.

. Geben Sie Einzelheiten zu der Datenbank an, die Sie in der <<Installationsvorbereitung>>
Kapitel. Klicken Sie dann auf _Speichern und fortfahren_.
+
[width="100%",frame="topbot",options="header"]
|================================
|Feldname | Erläuterung | Wert
|Datenbankname | Der der Datenbank gegebene benutzerdefinierte Name | drupal8
|Benutzername der Datenbank | Erstellter Benutzername | DatenbankBenutzername
|Datenbank-Passwort | Gewähltes Passwort | **************
|================================
+
--
// Go to the database configuration step of the installer
image:images/install-run-3.png["Database configuration form",width="100%"]
--

. Im nächsten Schritt wird ein Fortschrittsbalken unter der Überschrift _Installing
Drupal_. Nachdem das Installationsprogramm abgeschlossen ist, wechselt es automatisch zu
der nächste Schritt.
+
--
// Go to the page displaying the installation progress bar
image:images/install-run-4.png["Installation progress bar",width="100%"]
--

. Der nächste Schritt besteht darin, einige grundlegende Informationen über Ihre neue Website zu konfigurieren
(beachten Sie auch, wenn es eine Warnung über Dateiberechtigungen gibt, für einen späteren Schritt).
Beachten Sie, dass das Benutzerkonto, das Sie in diesem Schritt erstellen, der Administrator der Website ist.
Konto. Siehe <<<user-admin-account>> für wichtige Informationen über dieses einzigartige
Konto. Sie können dieses Konto getrost "admin" nennen, und stellen Sie sicher, dass Sie eine
sicheres und eindeutiges Passwort.
+
Füllen Sie das Formular mit den folgenden Informationen aus:
+
[width="100%",frame="topbot",options="header"]
|================================
|Feldname | Erläuterung | Wert
|Standortname | Der für den Standort gewählte Name | Anytown Farmers Market
|E-Mail-Adresse der Website | Die mit der Website verbundene E-Mail-Adresse | info@example.com
|Benutzername | Anmeldedaten des designierten Benutzers| admin
|Passwort | Das gewählte Passwort | **************
| Passwort bestätigen | Passwort wiederholen | ****************
|E-Mail-Adresse | E-Mail-Adresse des Benutzers | admin@example.com
|================================
+
Die übrigen Felder können wahrscheinlich auf ihren Standardwerten belassen werden.
+
--
// Go to the basic site information configuration step of the installer
image:images/install-run-5.png["Configuration form",width="100%"]
--

. Klicken Sie auf _Speichern und fortfahren_.

. Sie werden auf die Startseite Ihrer neuen Website umgeleitet, und Sie sollten
die Meldung _Glückwunsch, Sie haben Drupal installiert!_, die oben in
die Seite.
+
--
// Front page of Drupal after the installer just completed showing the success message
image:images/install-run-6.png["Installation success",width="100%"]
--

. Möglicherweise haben Sie im Konfigurationsschritt eine Warnung über Dateiberechtigungen gesehen,
und Sie können diese Warnung weiterhin sehen, bis Sie die Berechtigungen korrigiert haben. An
vermeiden Sie die Warnung und machen Sie Ihre Website sicherer, ändern Sie die Berechtigungen auf der
Verzeichnis _sites/default_ und die Datei _sites/default/settings.php_, so dass sie
schreibgeschützt sind (möglicherweise müssen Sie die Dokumentation Ihres gastgebenden Unternehmens konsultieren
wie man das macht).
