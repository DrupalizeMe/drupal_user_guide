=== Eine Rolle erstellen

[role="summary"]
Wie Sie eine neue Rolle erstellen.

(((User role,creating)))
(((Role,creating)))
(((Role,anonymous user)))
(((Role,authenticated user)))
(((Role,administrator)))

Zielsetzung

Erstellen Sie eine Anbieterrolle, um einigen - aber nicht allen - Benutzern zu ermöglichen, bestimmte
Aufgaben.

==== Erforderliche Vorkenntnisse

<<user-concept>>

// ==== Anforderungen an die Website

==== Schritte

. Navigieren Sie im Verwaltungsmenü _Verwaltung_ zu _Personen_ > _Rollen_
(_admin/people/roles_).

. Sie finden die Standardrollen _Anonymer Benutzer_,
_Authentifizierter Benutzer_, und _Administrator_ bereits vorhanden.
+
--
// Roles page (admin/people/roles).
image:images/user-new-role-roles-page.png["Roles page"]
--

. Klicken Sie auf _Rolle hinzufügen_, um eine benutzerdefinierte Rolle hinzuzufügen.

. Geben Sie im Feld _Rollenname_ den Typ Lieferant ein. Klicken Sie auf _Speichern_.
+
--
// Add role page (admin/people/roles/add).
image:images/user-new-role-add-role.png["adding vendor as a role"]
--

. Oben sehen Sie die Meldung "Role Vendor has been added".
der Seite.
+
--
// Confirmation message after adding new role.
image:images/user-new-role-confirm.png["vendor role added success message"]
--
