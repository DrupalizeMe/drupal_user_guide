[[structure-taxonomy-setup]]

=== Настройка таксономии

[role="summary"]
Как создать словарь таксономии и добавить его поле к типу материалов.

(((Таксономия,создание)))
(((Словарь,создание)))
(((Метаданные,создание)))
(((Список терминов,создание)))

==== Цель

Создать словарь Ингридиенты и добавить его в тип материала Рецепт
как поле, которое может содержать неограниченное количество значений и это позволит добавлять
новые термины в словаре.

==== Необходимые знания

* <<planning-data-types>>
* <<structure-taxonomy>>
* <<structure-fields>>

==== Требования к сайту

Тип материала Рецепт должен быть создан. Смотрите <<structure-content-type>>.

==== Шаги

. В _Управлении_ административного меню, перейдите в _Структура_ > _Таксономия_
(_admin/structure/taxonomy_). Вы увидите словарь _Теги_ vocabulary, который был
создан Стандартным установочным профилем. (Обратите внимание, что имя и
описание этого словаря показано на английском на этой странице; смотрите
<<language-concept>> для пояснения.)
+
--
// Taxonomy list page (admin/structure/taxonomy).
image:images/structure-taxonomy-setup-taxonomy-page.png["Страница таксономии",width="100%"]
--

. Нажмите _Добавить словарь_, и заполните значения ниже.
+
[width="100%",frame="topbot",options="header"]
|================================
| Название поля | Объяснение | Пример значения
| Название | Название словаря | Ингридиенты
| Описание | Краткое описание словаря | (Оставить пустым)
|================================
+
--
// Add Ingredients vocabulary from admin/structure/taxonomy/add.
image:images/structure-taxonomy-setup-add-vocabulary.png["добавление таксономии Ингридиенты",width="100%"]
--

. Нажмите _Сохранить_. Вы попадете на страницу Ингредиентов, которая показывает список
всех терминов в этом словаре.
+
--
// Ingredients vocabulary page
// (admin/structure/taxonomy/manage/ingredients/overview).
image:images/structure-taxonomy-setup-vocabulary-overview.png["Список терминов таксономии ингредиентов",width="100%"]
--

. Нажмите _Добавить термин_. Введите "Масло" в поле _Название_. Нажмите _Сохранить_.
+
--
// Name portion of Add term page
// (admin/structure/taxonomy/manage/ingredients/add).
image:images/structure-taxonomy-setup-add-term.png["Добавить страницу терминов",width="100%"]
--

. Вы увидите подтверждение, что создали термин. Добавьте больше
терминов. Например, "Яйца" и "Молоко".

. В _Управлении_ административного меню, перейдите в _Структура_ > _Типы материалов_
(_admin/structure/types_). Нажмите _Управление полями_ для вашего типа материала Рецепт.

. Нажмите _Добавить поле_, и введите значения из таблицы ниже. Нажмите _Сохранить и
продолжить_.
+
[width="100%",frame="topbot",options="header"]
|================================
|Название поля| Описание | Значение
| Добавить поле | Выбор типа поля | Ссылка > Термин таксономии
| Метка | Заголовок поля| Ингридиенты
|================================
+
--
// Add field page to add Ingredients taxonomy reference field.
image:images/structure-taxonomy-setup-add-field.png["Добавление поля Ингридиенты в тип материала Рецепт"]
--

. На следующем экране конфигурации, введите значения из таблицы ниже.
Click _Save field settings_.
+
[width="100%",frame="topbot",options="header"]
|================================
|Название поля| Описание | Значение
| Тип элемента для ссылки| Тип объекта, на который ссылается поле | Термин таксономии
| Допустимое количество значений | Количество значений, которые пользователь может ввести | Неограниченно
|================================
+
--
// Field storage settings page for Ingredients field.
image:images/structure-taxonomy-setup-field-settings.png["Настройки поля для Ингридиентов"]
--

. На следующем экране конфигурации, введите значения из таблицы ниже. Нажмите _Сохранить настройки_.
+
[width="100%",frame="topbot",options="header"]
|================================
|Название поля | Описание | Значение
|Справочный текст | Справка, показанная пользователям, создающим контент |Введите ингредиенты, которые посетители сайта могут искать
|Тип связи > Тип ссылки | Выберите метод, используемый для выбора допустимых значений | По умолчанию
|Тип связи > Словарь | Выберите словарь, чтобы выбрать из него допустимые значения | Ингредиенты
|Тип связи > Создать сущности по ссылке, если они еще не созданы | Можно ли создавать новые термины для ингредиентов из формы редактирования контента | Выбрано
|================================
+
--
// Reference type section of field settings page for Ingredients field.
image:images/structure-taxonomy-setup-field-settings-2.png["настройка поля ссылки",width="100%"]
--


. Нажмите _Сохранить настройки_. Вы вернетесь на страницу _Управление Полями_.
Появится сообщение о том, что конфигурация для ингредиентов
сохранена.
+
--
// Manage fields page showing Ingredients field on Recipe content type.
image:images/structure-taxonomy-setup-finished.png["Страница управления полями для типа материала Рецепт",width="100%"]
--
