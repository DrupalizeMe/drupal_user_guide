=== Установка модуля Drupal

[role="summary"]
Как установить модуль, используя административный интерфейс или Drush.

==== Цель

Установить модуль, файлы которого уже
загружены на сайт, через административный интерфейс или Drush.

==== Необходимые знания

<<understanding-modules>>

==== Требования к сайту

Если вы хотите использовать Drush для установки модулей, Drush должен быть установлен. Смотрите
<<install-tools>>.

==== Шаги

Вы можете использовать административный интерфейс или Drush для установки модулей.

===== Использование административного интерфейса

. В административном меню _Упаравление_, перейдите в _Расширение_
(_admin/modules_). На странице _Расширение_ показываются все доступные модули
на вашем сайте.

. Проставьте галочки для модуля или модулей, которые вы хотите установить. Например,
поставьте галочку для модуля ядра Activity Tracker.
+
--
// Top part of Core section of admin/modules, with Activity Tracker checked.
image:images/config-install-check-modules.png["Включение модуля Activity Tracker",width="100%"]
--

. Нажмите _Установить_. Выбранные модули будут установлены.

===== Использование Drush

. В разделе _Управление_ административного интерфейса, перейдите в _Расширения_
(_admin/modules_). На странице _Расширения_ показаны все установленные модули
на вашем сайте.

. Найдите машинное имя модуля вы хотите установить, раскрыв
блок информации для нужного модуля. Например, Activity Tracker
модуль имеет машинное имя _tracker_.

. Запустите следующую Drush команду, чтобы установить нужный модуль:
+
----
drush pm:enable tracker
----

==== Расширьте свое понимание

Если вы не наблюдаете изменений на вашем сайте, возможно вам нужно
почистить кэш. Смотрите <<prevent-cache-clear>>.
