=== Creating a User Account

(((User account,creating)))

==== Goal

Create vendor user accounts for Sweet Honey and Happy Farm vendors.

==== Prerequisite knowledge

* <<user-concept>>
* <<user-admin-account>>
* <<user-new-role>>

==== Site prerequisites

The Vendor role must exist on your site. See <<user-new-role>>.

==== Steps

. In the _Manage_ administrative menu, navigate to _People_.

. Click _Add user_.

. Fill in the form fields. See the table below.
+
[width="100%",frame="topbot",options="header"]
|================================
|Field name |Explanation |Example value
|Email address |A valid email address for the vendor. All emails from the system will be sent to this address. The email address is not made public. | info@example.com
|Username |A username for the vendor that they will use to sign in or author content items. Spaces are allowed; punctuation is not allowed except for periods, hyphens, apostrophes, and underscores. |Sweet Honey
|Password |A password the vendor will use to sign in to the site. You can see how safe the password is on the _Password strength_ gauge. You also get tips on how to make it safer.  |jYZgb678
|Confirm password |Type the same password to avoid any typing mistakes. |jYZgb678
|Status |Set the status of the user account. _Blocked_ users will not be able to sign in. |Active
|Roles |Set the role of the user account.|Vendor
|Notify user of new account |Whether or not to send a notification to the vendor's email address. |Checked
|Picture |Click _Browse_ and select a picture to upload. Pay attention to size restrictions. | Photo of the farm
|Contact settings |Enable or disable the display of a contact form for the account. | Checked
|Locale settings |Select the time zone for the user account from the select list. | America/Kentucky/Louisville
|================================
+
// This screenshot was taken from a default Drupal Basic user create form (/admin/people/create).
image:images/user-new-user_form.png["Form for creating a new user account",width="100%"]

[start=4]
. Click _Create new account_ to create the new account for the vendor. You will
get a notification about the creation of the new user account similar to the one
below.
+
// This screenshot was taken from a default Drupal Basic user create form (/admin/people/create) after adding a new user account by following the steps above.
image:images/user-new-user-created.png["New user account has been created",width="100%"]

[start=5]
. Create a second vendor account for Happy Farm, following the steps mentioned
above.

==== Follow-on tasks

Create a user account for yourself.
