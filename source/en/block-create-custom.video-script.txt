=== Creating A Custom Block

(Slide/Intro)
This tutorial covers how to create a custom block.

(Slide/Goal)
By the end of this tutorial you'll know how to create a block showing the hours and location of the farmers market.

(Slide, Prerequisites)
Before you get started with this tutorial you'll want to make sure your familiar with

- Know what blocks are, and the relationship between blocks and regions.

See the written version of this tutorial for links to each of these prerequisite tutorials.

(Demo/Browser)

In the _Manage_ administrative menu, navigate to _Structure_ > _Block layout_ > _Custom block library_ (_admin/structure/block/block-content_).

Click _Add custom block_. and wait until the _Add custom block_ page appears.

To create a new custom block fill in the provided form. For the farmers market we need an hours and location block and we can fill in the form with that information:

The block description field contains the name of the block as we want it shown to administrators. Value ="Hours and location block"

For the body field add in the content you would like to display in the block. Value = "Open: Sundays, 9 AM to
2 PM, April to September<br />Location: Parking lot of Trust Bank, 1st & Union,
downtown Anytown."

After filling in the form click _Save_. A message should appear indicating the block has been saved.

(Slide/Recap)
In this tutorial we:

- Created a new custom block that when placed into a region on our site will display the hours and location for the farmers market.
