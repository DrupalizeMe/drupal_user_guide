[[structure-content-type-delete]]

=== Deleting a Content Type

(((Structure Content Type,deleting)))

==== Goal

Learn about deleting unwanted content types. Here in our example let us delete the 'Article' content type which may not be useful for us.

==== Prerequisite knowledge

* <<config-overview>>

//==== Site prerequisites

==== Steps

. Visit Menu > Structure then click on 'content types' as shown in the below
  figure:

// To create this image, install Drupal 8 successfully, visit Menu > Structure then click on 'content types' and get the screenshot
image:images/structure-content-type-delete-listing.png["content-types",width="100%"]

. There you can see list of all the available types of content having a dropdown
  available on the right side (under Operations heading)for every content
  type. Click on the dropdown against the unwanted content type (which you wish
  to delete from the system - here 'Article' in our example) and click on
  'Delete' option as shown in the below figure (considering deletion of
  'Article' content type).

// To create this image, install Drupal 8 successfully, visit Menu > Structure
// then click on 'content types' and click on the dropdown against 'Article'
// content type and get the screenshot
image:images/structure-content-type-delete-dropdown.png["Article-dropdown",width="100%"]

. There is a confirmation page displayed with all the basic default
  configurations listed as shown in the below figure: Click on 'Delete' button

// To create this image, install Drupal 8 successfully, visit Menu > Structure
// then click on 'content types' and click on the dropdown against 'Article'
// content type, click on 'delete' option and get the screenshot
image:images/structure-content-type-delete-confirmation.png["Article-delete",width="100%"]

. Content type listing page appears with a confirmation message displayed on the
  top of it as shown in the below figure:

// To create this image, install Drupal 8 successfully, visit Menu > Structure
// then click on 'content types' and click on the dropdown against 'Article'
// content type, click on 'delete' option, click on 'delete' button and get
// the screenshot
image:images/structure-content-type-delete-confirm.png["Article-delete-confirm",width="100%"]
