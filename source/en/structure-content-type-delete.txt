[[structure-content-type-delete]]

=== Deleting a Content Type

[role="summary"]
How to delete a content type that is not required for the site.

(((Content type,deleting)))

==== Goal

Delete the unneeded content type _Article_.

==== Prerequisite knowledge

<<config-overview>>

==== Site prerequisites

The _Article_ content type must exist. It is created on your site when you
install with the core Standard installation profile.

==== Steps

. In the _Manage_ administrative menu, navigate to _Structure_ > _Content
types_ (_admin/structure/types_). The _Content types_ page appears.

. Click _Delete_ in the _Operations_ dropdown button for the _Article_ content
type.
+
--
// Content types list on admin/structure/types, with operations dropdown
// for Article content type expanded.
image:images/structure-content-type-delete-dropdown.png["Article delete dropdown",width="100%"]
--

. A confirmation page is displayed. Click _Delete_.
+
--
// Confirmation page for deleting Article content type.
image:images/structure-content-type-delete-confirmation.png["Article delete confirmation page",width="100%"]
--

. The _Content types_ page appears with a confirmation message saying that the
content type has been deleted:
+
--
// Confirmation message after deleting Article content type.
image:images/structure-content-type-delete-confirm.png["Article delete confirmation message"]
--
