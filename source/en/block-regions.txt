==== What is a region?

Besides its primary content, a web page contains site branding (site name,
slogan, and logo) , navigation aids (menus, links, and icons), formatted text,
images, and like items. Each theme provides a set of named regions, such as
_Header_, _Content_, and _Sidebar_, where site builders may choose to place
such items.

The available regions depend on the theme design. Only the _Content_ region,
which contains the primary content, is required; others are optional.  The
core Bartik theme provides the eighteen regions highlighted in the following
image.
